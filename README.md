<h1 style="text-align: center">Common Ground</h1>

Common Ground is an app that encourages real-world activities between people with shared interests.

It allows you to schedule events in public or private locations.

#### Examples of events
- Fan convention
- Chess competition
- Friends party
- Book swapping or book exchange
- Marathon
- Restaurant grand opening

# Features
- Create and share your event as an admin with properties such as description, location, time, tags, public/private, attendee limit (if private)...etc.
- Users who are interested in coming will join a temporary group chat to discuss the details of the event and get hyped about it.
